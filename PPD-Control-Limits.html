Empirical Bayes Posterior Predictive Intervals for Funnel Plots

Andrew Johnson

12/01/2021

Funnel Plot Control Limits

Prediction intervals are (generally) assumed to be constructed in the Frequentist framework, which frames probability in terms of long-run frequency. Alternatively, these intervals can be constructed in a Bayesian framework, which allows for the natural use of distributions that can account for overdispersion in the observed data.

Overdispersion is a common issue encountered in the construction of control limits. Overdispersion occurs when the observed data varies more than would be expected by the distribution being used to model the data. To account for this, the data can be modelled using distributions that explicitly account for the overdispersion. The choice of distribution will depend on the type of data being modelled.

Funnel Plot Data

There are two (primary) types of data assessed using funnel plots: proportions and counts.

Proportions

Binomial Distribution & Overdispersion

Traditionally, proportion data is modelled using the Binomial distribution, given by: \[ y \sim Bin(N,\theta) \] Where \(y\) is the number of complaints successfully resolved, \(N\) is the total number of complaints made, and \(\theta\) the probability of successfully resolving a complaint. The binomial distribution implies that the outcome (\(y\)) has variance: \[ N*\theta*(1-\theta) \]

If the observed response \(y\) has a variance greater than \(N*\theta*(1-\theta)\), then we would conclude that the data is overdispersed. To account for this, we can introduce an additional dispersion parameter that scales the variance. This can be achieved using a Beta-Binomial distribution.

Beta-Binomial

The Beta-Binomial distribution again models the observed proportions as a binomial distribution, but instead the probability of success is assumed to follow a Beta distribution: \[ y \sim Bin(N,\theta) \]

\[ \theta \sim Beta(\alpha, \beta) \]

The hyperparameters \(\alpha\) and \(\beta\) can then be used to model the overdispersion in the data, as they allow for a scaling of the observed variance. These hyperparameters are commonly selected using maximum-likelihood estimation.

To generate prediction intervals for the beta-binomial distribution using these estimated hyperparameters, we can then use Bayesian posterior predictive distributions. This process of estimating hyperparameters from the observed data is also known as Empirical Bayes.

Posterior Predictive Distribution

Where the posterior distribution refers to the combination of the observed likelihood and the specified priors, the posterior predictive distribution refers to the distribution of new observations, given the current posterior distribution (i.e., \(P(y_{new}|y)\)).

As posterior predictive distributions can be used to derive the mean and standard deviation (SD) of new observations, we can use these to derive control limits. In other words, we would expect 99.7% of new observations to fall within 3SD of the mean: \[ E[y_{new}|y] \pm 3 *\sqrt{Var(y_{new}|y)} \]

Beta-Binomial Prediction Intervals

Given the Beta-Binomial model: \[ y \sim Bin(N,\theta) \]

\[ \theta \sim Beta(\alpha, \beta) \]

The resulting posterior distribution is given by: \[ \theta|y \sim Beta(\alpha + y, \beta + N-y) \] If we then denote the hyperparameters of this distribution as: \[ \alpha_1 = \alpha + y \] \[ \beta_1 = \beta + N-y \] The posterior predictive distribution of the \(y_{new}\) successes, given an additional \(m\) trials, is also given by the Beta-Binomial distribution: \[ y_{new} \sim BetaBin(m,\alpha_1, \beta_1) \]

If we denote the mean parameter: \[ \theta_1 = \frac{a_1}{a_1+b_1} \] And dispersion parameter: \[ \phi = \frac{1}{a_1 + b_1 + 1} \]

The mean of this distribution (i.e., the average number of new successes, \(y_{new}\), given \(m\) new trials) is then given by: \[ E[y_{new}|y] = m \theta_1 \]

And the variance given by: \[ Var(y_{new}|y) = m\theta_1(1-\theta_1)[1+\phi(m-1)] \]

Note that as the dispersion parameter approaches 0, the variance of the beta-binomial distribution approaches the binomial distribution. In other words, the beta-binomial allows for a scaling of the binomial variance.

By using the beta-binomial distribution, we can construct control limits for the ranges of observations we would expect if the new values were as overdispersed as those currently observed. Additionally, we could also scale this expected overdispersion, to account for the possibility that the observed overdispersion could in fact be an underestimate of the true overdispersion.

The control limits for the funnel plot are then derived by estimating these intervals for each value of \(m\) on the x-axis.

Worked Example

First well load the packages and a dataset for analysis:

library(tidyverse)

dat = read.csv("V0014.csv")

Next well construct the parameters for the posterior predictive distribution:

y = sum(dat$numerator)
N = sum(dat$denominator)

# Estimate optimal alpha and beta parameters to account for overdispersion
tmp_fit = as.numeric(VGAM::Coef(VGAM::vglm(cbind(numerator,denominator-numerator) ~ 1, 
                                           VGAM::betabinomialff,
                                           data = dat)))

alpha = tmp_fit[1]
beta = tmp_fit[2]

Then well specify a function to give the predicted mean and variance of successes given a number of new trials:

mv_pred = function(m, y, N, alpha, beta, disp_scale) {
  a_1 = alpha + y
  b_1 = beta + N - y
  theta_1 = a_1 / (a_1 + b_1) 
  phi = (1 / (a_1 + b_1 + 1) ) * disp_scale
  pred_mean = m * theta_1
  pred_var = m * theta_1 * (1 - theta_1) * (1 + phi * (m - 1))
  return(c(pred_mean,pred_var,phi))
}

Note that the function has the input argument disp_scale (which will be given the default value of 1 in the next function). This will be used to scale the amount of overdispersion when constructing control limits, which will be covered shortly.

And a function that will use the predicted mean and variance to construct 95% and 99% intervals:

lims = function(m,y,N,alpha,beta,disp_scale=1){
  params = mv_pred(m,y,N,alpha,beta,disp_scale)
  pred = params[1]
  sd = sqrt(params[2])
  phi = params[3]
  data.frame(
    x = m,
    phi = phi,
    # Truncate lower limits at 0
    #  i.e., can't have negative successes
    LCI_99 = max(pred - 3*sd,0) / m,
    LCI_95 = max(pred - 2*sd,0) / m,
    # Truncate upper limits at m
    #  i.e., can't have more successes than trials
    UCI_95 = min(pred + 2*sd,m) / m,
    UCI_99 = min(pred + 3*sd,m) / m
  )
}

Now we can generate the intervals for a range of denominator values:

cnt_lims = map_dfr(seq.int(1,max(dat$denominator) + 10),lims,y,N,alpha,beta)

And plot the result:

overall_prop = y/N

dat %>%
  mutate(prop = numerator / denominator) %>%
  select(x=denominator,prop) %>%
  left_join(cnt_lims,.) %>%
    ggplot(aes(x=x,y=prop)) + 
      geom_point(na.rm=T) + 
      geom_segment(aes(y = overall_prop, yend = overall_prop,
                       x = min(x), xend = max(x))) +
      geom_line(aes(x=x,y=LCI_99,linetype="3SD")) + 
      geom_line(aes(x=x,y=UCI_99,linetype="3SD")) +
      geom_line(aes(x=x,y=LCI_95,linetype="2SD")) + 
      geom_line(aes(x=x,y=UCI_95,linetype="2SD")) +
      scale_linetype_manual(name="Intervals", values=c("dashed","solid")) +
      theme_bw()

We can then compare this with the current best-practice funnel-plot (given by the FunnelPlotR package):

FunnelPlotR::funnel_plot(numerator=dat$numerator, denominator=dat$denominator,
                         group=dat$agg_group_entity, data_type = "PR",
                         OD_adjust = TRUE, label_outliers = FALSE)

Overall, the empirical bayes posterior-predictive approach appears to handle small counts better but the intervals are narrower with larger counts. This suggests that the approach is more conservative in its modelling of overdispersion. Given this, we could also allow for a greater amount of overdispersion in our construction of the control limits. As an arbitrarily chosen figure, lets allow for the control limits to be twice as overdispersed as the observed data. For this well use the disp_scale argument mentioned earlier:

cnt_lims_od = map_dfr(seq.int(1,max(dat$denominator) + 10),
                      lims,y,N,alpha,beta,disp_scale=2)

dat %>%
  mutate(prop = numerator / denominator) %>%
  select(x=denominator,prop) %>%
  left_join(cnt_lims_od,.) %>%
    ggplot(aes(x=x,y=prop)) + 
      geom_point(na.rm=T) + 
      geom_segment(aes(y = overall_prop, yend = overall_prop,
                       x = min(x), xend = max(x))) +
      geom_line(aes(x=x,y=LCI_99,linetype="3SD")) + 
      geom_line(aes(x=x,y=UCI_99,linetype="3SD")) +
      geom_line(aes(x=x,y=LCI_95,linetype="2SD")) + 
      geom_line(aes(x=x,y=UCI_95,linetype="2SD")) +
      scale_linetype_manual(name="Intervals", values=c("dashed","solid")) +
      theme_bw()

However, there is very little change in the control limits. If we inspect the (unscaled) dispersion parameter:

cnt_lims$phi[1]
## [1] 0.0009107578

We can see that there is very little overdispersion, and so the observations exceeding the control limits are likely to be outliers, rather than the result of unmodelled dispersion.

We can take a similar approach to the modelling of count data.

Count Data

Poisson Distribution

Counts, and their prediction intervals, are commonly modelled using the Poisson distribution: \[ y \sim Poisson(\lambda) \]

The Poisson distribution implies that the observations have equal mean and variance: \[ \lambda = E(y) = Var(y) \]

If the variance is greater than the mean, the data are said to be overdispersed.

Negative-Binomial Distribution

A alternative distribution for modelling overdispersed count data is the Negative-Binomial: \[ y \sim NB(r, \frac{p}{p+r}) \]

Where \(r\) models the extent of dispersion, and \(p\) is the average number of counts for the \(N\) observed units. The mean of the distribution is given by: \[ E(y) = p \]

The variance of this distribution is given by: \[ Var(y) = p + \frac{p^2}{r} \]

As such, the additional dispersion parameter allows the variance of the observations to be greater than the mean. Note also that as the dispersion parameter \(r\) approaches \(\infty\), the negative binomial distribution approaches the poisson. As the control limits will be constructed for a range of denominator values, we only need the dispersion parameter \(r\). As such, the expectation parameter \(p\) is fixed to 1.

Similar to the Beta-Binomial model above, the hyperparameter \(r\) can be estimated using maximum-likelihood and passed to the posterior predictive distribution.

Negative-Binomial Prediction Intervals

To generate prediction intervals well take advantage of the fact that the negative-binomial distribution: \[ y \sim NB(r, \frac{p}{p+r}) \] can be parameterised as a poisson-gamma mixture: \[ y \sim Poisson(\lambda) \] \[ \lambda \sim Gamma(r,\frac{p}{r}) \]

The posterior-predictive distribution of the Poisson-Gamma mixture is itself a Negative-Binomial, with mean: \[ E[y_{new}|y] = \frac{\Sigma x_i + r}{N + \frac{p}{r}} \] and variance: \[ Var[y_{new}|y] = \frac{\Sigma x_i + r}{(N + \frac{p}{r})^2}(N+\frac{p}{r}+1) \] However, because were deriving prediction intervals for the observation of one count of size \(x\) (i.e., one for each value of the funnel plot x-axis), the above forms are simplified. As mentioned above, the \(p\) parameter is simply 1, \(\Sigma x_i\) reduces to \(x\) and \(N\) is simply 1.

This gives the revised posterior predictive mean: \[ E[y_{new}|y] = \frac{x + r}{1 + \frac{1}{r}} \]

And variance: \[ Var[y_{new}|y] = \frac{x + r}{(1 + \frac{1}{r})^2}(1+\frac{1}{r}+1) \]

Worked Example

We will again load a dataset for testing

dat = read.csv("V0013.csv")

First well estimate the dispersion parameters for the Negative-Binomial distribution:

tmp_fit = as.numeric(VGAM::Coef(VGAM::vglm(numerator ~ 1, VGAM::negbinomial(), data=dat)))
disp = tmp_fit[2]

Next we need a function to calculate the mean and variance of the posterior-predictive distribution for a given count:

mv_pred = function(x,disp,disp_scale){
  phi = disp / disp_scale
  a = phi
  b = 1 / phi
  m = (x+a)/(1+b)
  v = ((x+a)/(1+b)^2) * (1+b+1)
  return(c(m,v,phi))
}

Note that the amount of dispersion is scaled by dividing the dispersion parameter (rather than multiplying, as in the beta-binomial), as smaller values indicate greater dispersion.

And a function that will use that mean and variance to calculate the control limits.

lims = function(x,disp,disp_scale=1) {
  params = mv_pred(x,disp,disp_scale)
  m = params[1]
  s = sqrt(params[2])
  phi = params[3]
  data.frame(
      x = x,
      phi = phi,
      # Truncate lower limits at 0
      #  i.e., can't have negative counts
      "LCI_99" = max(m - 3*s,0) / m,
      "LCI_95" = max(m - 2*s,0) / m,
      "UCI_95" = (m + 2*s) / m,
      "UCI_99" = (m + 3*s) / m
  )
}

We can then generate the control limits for the range of x-values:

cnt_lims = map_dfr(seq.int(0,max(dat$denominator) + 10),lims,disp)

And plot the results:

ggplot(dat,aes(x=denominator,y=(numerator/denominator))) + 
  geom_point(na.rm=T) + 
  geom_segment(aes(y = 1, yend = 1,
                   x = min(denominator), xend = max(denominator) + 10)) +
  geom_line(data=cnt_lims,mapping=aes(x=x,y=LCI_99,linetype="3SD")) + 
  geom_line(data=cnt_lims,mapping=aes(x=x,y=UCI_99,linetype="3SD")) +
  geom_line(data=cnt_lims,mapping=aes(x=x,y=LCI_95,linetype="2SD")) + 
  geom_line(data=cnt_lims,mapping=aes(x=x,y=UCI_95,linetype="2SD")) +
  scale_linetype_manual(name="Intervals", values=c("dashed","solid")) +
  scale_y_continuous(name="Observed/Expected",limits = c(0,max((dat$numerator/dat$denominator))+0.5)) +
  theme_bw()

We can then compare the results of the funnel plot generated by the FunnelPlotR package:

FunnelPlotR::funnel_plot(numerator=dat$numerator, denominator=dat$denominator,
                         group = dat$agg_group_entity, data_type = "SR", Poisson_limits = FALSE,
                         OD_adjust = TRUE, sr_method = "CQC", label_outlier=FALSE, limit=99)

We can see that the posterior predictive intervals provided dramatically different results to the FunnelPlotR intervals. This likely due to high levels of overdispersion in the data. If we inspect the dispersion parameter:

cnt_lims$phi[1]
## [1] 0.2775641

Noting that \(\phi = \infty\) results in the Poisson distribution, this small value indicates large overdispersion, which appears better handled by the posterior predictive intervals.
